## Installation

using `yarn` :

```
yarn add babel-plugin-array-last-index
```

using `npm` :

```
npm install babel-plugin-array-last-index --save
```

## Usage

```
// .babelrc
{
  "plugins": [
    "babel-plugin-array-last-index"
  ],
}

```
